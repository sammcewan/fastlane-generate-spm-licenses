# generate_spm_licenses plugin

[![fastlane Plugin Badge](https://rawcdn.githack.com/fastlane/fastlane/master/fastlane/assets/plugin-badge.svg)](https://rubygems.org/gems/fastlane-plugin-generate_spm_licenses)

## Getting Started

This project is a [_fastlane_](https://github.com/fastlane/fastlane) plugin. To get started with `fastlane-plugin-generate_spm_licenses`, add it to your project by running:

```bash
fastlane add_plugin generate_spm_licenses
```

You'll need to specify the Git URL of this repo.

## About generate_spm_licenses

Generates SPM licences assuming the product is correctly checked out.

## Example

Check out the [example `Fastfile`](fastlane/Fastfile) to see how to use this plugin. Try it by cloning the repo, running `fastlane install_plugins` and `bundle exec fastlane test`.

Look at the example project which shows how to use the script and set up acknowledgements within settings.

## Run tests for this plugin

To run both the tests, and code style validation, run

```
rake
```

To automatically fix many of the styling issues, use
```
rubocop -a
```
